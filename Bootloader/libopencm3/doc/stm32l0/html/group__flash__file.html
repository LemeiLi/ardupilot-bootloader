void flash_set_ws ( uint32_t  ws)

Set the Number of Wait States.

Used to match the system clock to the FLASH memory access time. See the programming manual for more information on clock speed and voltage ranges. The latency must be changed to the appropriate value before any increase in clock speed, or after any decrease in clock speed. A latency setting of zero only applies if 64-bit mode is not used.

Parameters
[in]wsvalues from FLASH Wait States.
